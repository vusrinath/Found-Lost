üì± User Flow Summary

Primary Flow

  1. User opens app to Home screen
  2. Taps "+" FAB to create new box
  3. Fills in box details and category
  4. App generates unique QR code
  5. User prints and attaches QR to physical box
  6. User adds items to digital box
  7. Items are searchable across all boxes

Search Flow

  1. User needs to find an item
  2. Types item name in search bar
  3. Results show both matching boxes and items
  4. Taps result to view box location
  5. User can navigate to physical box

QR Scan Flow

  1. User finds physical box
  2. Opens app and taps Scan tab
  3. Points camera at QR code
  4. App instantly opens box details
  5. User can view all items inside
  6. Can add/edit items on the spot

üé® Design Notes

  • Color Scheme: Purple gradient (#667eea to #764ba2) for modern, premium feel
  • iOS Design: Following Apple's Human Interface Guidelines with rounded corners, SF Symbols style icons
  • Tab Navigation: Bottom tabs for easy thumb access on large phones
  • FAB Button: Prominent floating action button for quick box creation
  • Card-Based: Box cards provide visual hierarchy and are easy to scan
  • Stats Dashboard: Quick overview of storage situation at a glance
